# RestaurangRegistret
En liten program för att kontrollera en restaurag registret.

## Bakgrund
1. Registret ska kunna ta emot och lagra anställda med namn och lön. (via inmatning i konsolen, ingen krav på persistent lagring)
2. Programet ska kunna skriva registret i en konsol.
